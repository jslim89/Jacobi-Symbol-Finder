# Description
This is a simple program to find out [Jacobi Symbol] (http://en.wikipedia.org/wiki/Jacobi_symbol) using the 4 properties

# How to use?
Basically you only need to input Numerator and Denominator
i.e.  Numerator    7411
     ----------- = ----
     Denominator   9283

     The final result will be -1
